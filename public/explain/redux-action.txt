Bước 1 : Tạo action
    redux => create file name ActionTypes.js
        ActionTypes.js {nơi thực hiện các Redux Action và những action creater}
            export 1 hằng số ADD_COMMENT và xuất 1 chuỗi 'ADD_COMMENT'
                => xuất nhiều hằng số, mỗi hằng số gắn với 1 string và là 1 hành động tương ứng
                => đây là nơi có thể nhập các loại hành động này vào các chức năng reduive và sau đó
                sử dụng nó để thực hiện kết hợp trong lệnh switch mà bạn sử dụng ở đó
                => đó là lý do tạo các tập tin ở đây
    redux => create file name ActionCreators.js
        ActionCreator file ( nơi chuẩn hóa tất cả ActionType vào ActionCreator)
            => import tất cả mọi thứ được xuất bên ActionTypes
            export 1 function addComment ( 1 hàm tạo ra 1 action object) trả về 1 object
                4 tham số : dishId, rating, author, comment
                    => đây cũng là những giá trị cần được truy xuất trong function này
                type (kiểu action) : điều đầu tiên cần define
                    => nơi dựng các kiểu action ở đây, có thể dễ dàng define type để Redux
                    ActionTypes.ADD_COMMENT
                payload (tải trọng)
                    dishId : dishId,
                    rating: rating,
                    author: author,
                    comment: comment
                    => ánh xạ chúng vào 4 thuộc tính bên trong payload < 4 thuộc tính này sẽ dk xử lý >
                gửi action này đến store
